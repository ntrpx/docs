# Ashwagandha (KSM-66)

<Note>
This ingredient monograph is under development.
</Note>

## Overview

Ashwagandha is a calming adaptogen that reduces cortisol by ~27% via 11β-HSD1 inhibition. Supports stress resilience, reduces anxiety, and may support testosterone levels.

## Taxonomy Position

- **Domain:** Adaptogens & Herbal Modulators / Endocrine & Hormonal
- **Category:** Classic Adaptogens (4.1), HPA Axis (3.1)
- **Subcategory:** Cortisol Modulators

## Key Properties

| Property | Effect |
|----------|--------|
| Cortisol | ↓ 27% |
| Anxiety | ↓ Significant |
| Testosterone | ↑ Modest |
| Character | Calming adaptogen |
| Active Compounds | Withanolides |

## Extract Standardization

- **KSM-66:** Full-spectrum root extract, 5% withanolides
- **Sensoril:** Root + leaf extract, higher withanolide content
