## Key Design Decisions

| Decision | Rationale |
|:---------|:----------|
| **CDP-Choline (not Alpha-GPC) daily** | Alpha-GPC: 46% stroke risk with 10+ years; CDP provides choline + uridine |
| **Alpha-GPC in Sprint™ only** | Occasional use avoids chronic risk; superior acute delivery |
| **Paraxanthine (not caffeine) daily** | 2.2× A2A affinity; no anxiogenic metabolites; faster clearance |
| **Caffeine in Sprint™ only** | Broader acute effect needed; sleep less relevant for single events |
| **Glycine at exactly 3g** | Threshold for thermoregulatory sleep onset (↓0.3°C core temp) |
| **No melatonin** | Suppresses endogenous production; creates dependency; glycine superior |
| **L-5-MTHF (not folic acid)** | 30%+ MTHFR variants; folic acid masks B12 deficiency; active form |
| **P5P (not pyridoxine)** | Pyridoxine neurotoxicity documented; P5P is active coenzyme |
| **L-Citrulline (not L-Arginine)** | 80% vs 20% bioavailability; bypasses arginase |
| **NR + Apigenin evening** | Apigenin inhibits CD38 → preserves NR-derived NAD+ |

## B-Vitamin Forms — Non-Negotiable

| Vitamin | ✅ Include | ❌ Exclude | Why |
|:--------|:-----------|:-----------|:----|
| **B1** | Benfotiamine, TTFD | Thiamine HCl (OK for sweat) | Higher bioavailability; CNS penetration |
| **B2** | Riboflavin-5'-Phosphate | Riboflavin | Active coenzyme |
| **B6** | Pyridoxal-5'-Phosphate | **Pyridoxine** | **Neurotoxicity risk** |
| **B9** | L-5-MTHF (Quatrefolic®) | **Folic acid** | **MTHFR variants; B12 masking** |
| **B12** | Methylcobalamin + Adenosylcobalamin | **Cyanocobalamin** | **Cyanide moiety; requires conversion** |
