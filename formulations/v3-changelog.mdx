# NTRPX Formulations v3 — Corrections & Refinements

## Critical Changes

### 1. Neuraldrink™ Sol — Removed Paraxanthine
- **v2:** Sol™ contained 100mg paraxanthine  
- **v3:** Sol™ is stimulant-free hydration + brain fuel
- **Rationale:** ParaCaffeine +™ is the optional stimulant add-on; users should be able to hydrate without stimulation

### 2. Salidroside — Premium Form Only
- **v2:** "3% salidroside Rhodiola" extract  
- **v3:** ≥98% precision-fermented pure salidroside
- **Dose:** Boost™ 15mg, Sprint™ 30mg
- **Placement:** Capsules ONLY (bitter — not suitable for drinks)

### 3. Huperzine A — Premium Form Only
- **v2:** "1% Huperzine A extract"  
- **v3:** ≥98% synthetic pure Huperzine A
- **Dose:** 100 mcg (not 200 mcg)
- **Rationale:** Pharmaceutical purity, exact dosing, sustainable

### 4. Melatonin — Added to Recover™
- **v2:** No melatonin (avoided due to dependency concerns)  
- **v3:** 0.3 mg physiological micro-dose
- **Evidence:** MIT research confirms 0.3mg is physiological, sustainable, and MORE effective than 3mg+ doses
- **Mechanism:** Complements glycine thermoregulation; different pathways

### 5. Sprint™ — Removed Non-Acute Ingredients
- **v2:** Included Phosphatidylserine (100mg) and Lion's Mane (500mg)  
- **v3:** Removed both
- **Rationale:** 
  - PS monograph states: "PS benefits build over time... this is not an acute-effect supplement"
  - Lion's Mane acute evidence is mixed; primary NGF mechanism is chronic
  - Lion's Mane moved to Boost™ (daily use)

### 6. Sodium Sources — Pharmaceutical Grade
- **v2:** "Sodium Citrate + Pink Himalayan 1,000 mg" (confusing)  
- **v3:** 
  - Sodium Citrate (USP) — 500mg
  - Sodium Chloride (USP) — 500mg
  - Total sodium: 1,000mg from pharmaceutical sources
- **Rationale:** Pink Himalayan salt has documented heavy metal (lead) and microplastic contamination

### 7. Neuraldrink™ — Added Carbohydrate System
- **v2:** Missing glucose/fructose formulation  
- **v3:** 
  - Sol™: 15g glucose + 8g fructose (65:35 ratio, 23g total)
  - Luna™: 8g glucose + 4g fructose (67:33 ratio, 12g total)
- **Rationale:** Brain glucose facilitation is core to Neuraldrink™ concept

### 8. Taste Profile Verification
Confirmed all Neuraldrink™ ingredients pass taste/solubility requirements:

| ✓ Included in Drinks | ✗ Excluded (Capsules Only) |
|:--------------------|:---------------------------|
| Glucose, Fructose | Salidroside (bitter) |
| Glycine (sweet) | PQQ (color, bitter) |
| Taurine (slightly sweet) | Alpha-GPC (fishy) |
| L-Citrulline (tart, citrus-compatible) | Huperzine A (extremely bitter) |
| L-Theanine (mild umami, maskable) | Phosphatidylserine (fat-soluble) |
| Magnesium Citrate (tart) | Bacopa (bitter) |
| B-vitamins (mild) | Ashwagandha (earthy) |
| Cordyceps water extract (mild) | Creatine (sandy, won't dissolve) |

### 9. All Systems Go™ Independence
Verified that All Systems Go™ provides complete daily optimization WITHOUT requiring Neuraldrink™:
- ✓ Full creatine (5g/day split Boost + Sustain)
- ✓ Full choline (550mg/day CDP-Choline)
- ✓ Full B-complex (active forms)
- ✓ Full omega-3 (DHA + EPA)
- ✓ Full sleep support (glycine, Mg, NR, melatonin)
- ✓ Full adaptogens (salidroside, ashwagandha, bacopa)

Neuraldrink™ ADDS hydration + brain glucose + additional sleep support — but is not required.
